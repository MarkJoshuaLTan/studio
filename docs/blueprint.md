## Core Features:

- Real Property Tax Calculation: Calculate real property tax based on location, lot area, and property type.
- Omnibox Autocomplete: Implement an autocomplete dropdown for location and barangay fields to enhance user input speed and accuracy. The LLM will act as a tool to extract relevant values from existing values based on user input.
- Light/Dark Mode Toggle: Add a toggle switch for users to switch between light and dark mode for improved accessibility and user preference.
- 2028 Estimated Tax: Display the estimated yearly tax for 2028, using provided values.
- 2028 Real Yearly Tax: Calculate the real yearly tax for 2028, by adding 6% to the 2028 estimated tax. Display the new value.

## Style Guidelines:

- Background color: White (#FFFFFF) for a clean, modern look.
- Primary color: Forest green (#228B22) for a natural, trustworthy feel.
- Accent color: Lime green (#32CD32) to highlight important interactive elements and call-to-actions.
- Body and headline font: 'Inter' sans-serif font for clear and modern readability.
- Use simple, geometric icons to represent property types and settings.
- Maintain a clean, well-spaced layout to improve user experience.
- Incorporate subtle transition animations when switching between light and dark mode.
